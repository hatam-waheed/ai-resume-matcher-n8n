# AI Resume Matcher and ATS

## Overview

AI Resume Matcher is an automated Applicant Tracking System built using n8n and Google Gemini. It evaluates resumes against job descriptions, calculates candidate-job match scores, identifies missing skills, and provides personalized improvement suggestions.

---

## Features

✅ Resume Upload

✅ PDF Text Extraction

✅ Job Description Analysis

✅ AI Match Score Calculation

✅ Skill Gap Detection

✅ Resume Improvement Suggestions

✅ Google Sheets Logging

---

## Workflow

### Step 1: Candidate Submission

The candidate submits:

* Name
* Email
* Job Title
* Job Description
* Resume PDF

### Step 2: Resume Extraction

The uploaded PDF is converted into raw text.

### Step 3: AI Evaluation

Google Gemini analyzes:

* Resume content
* Job requirements
* Candidate skills

The AI calculates:

* Match Score
* Matched Skills
* Missing Skills
* Resume Suggestions

### Step 4: Data Formatting

The workflow formats AI-generated lists into readable text.

### Step 5: Spreadsheet Logging

Results are automatically saved into Google Sheets.
